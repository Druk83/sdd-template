## Контекст
- Компонент/сервис: `.tools/plantuml-render/plantuml_render.py`, документация рендера диаграмм
- Версия/коммит: <указать>
- Окружение: Windows/macOS/Linux (Docker доступен)
- Частота: постоянно при генерации диаграмм
- Impact: сейчас диаграммы отправляются во внешний сервис; нужен локальный рендер через Docker

## Требования
- Локальный Kroki поднимается через Docker/Compose
- Скрипт рендера умеет использовать локальный URL (например, `http://localhost:8000`)
- Документация описывает запуск и переменные окружения

## Ожидаемое поведение
Диаграммы рендерятся через локальный Kroki без выхода в интернет.

## Фактическое поведение
Используется внешний сервис `https://kroki.io/`.
